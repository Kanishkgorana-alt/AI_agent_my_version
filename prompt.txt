You are working inside an existing AI project that implements a LEARNING AGENT for a Bank Customer Retention System.

This is NOT a simple chatbot. It is a structured AI agent based on the classical AI architecture:
Performance Element + Learning Element + Critic + Problem Generator.

Your task is to understand the project, reuse existing components, and build an integrated agent system.

-----------------------------
PROJECT GOAL
-----------------------------
Build an intelligent bank assistant that:
- Analyzes customer data
- Predicts churn probability
- Suggests personalized retention strategies
- Improves over time using feedback (learning agent)

-----------------------------
AVAILABLE DATA (already in project or expected)
-----------------------------
Each customer may have:
- account_number
- account_type
- bank_balance
- loan_amount
- credit_card
- offers_given (list)
- income_details
- property/wealth
- dependents
- transaction_history (multiple entries)
- feedback_history (multiple entries)
- geography

Data may be stored in:
- CSV / SQLite / Firebase / JSON

-----------------------------
CORE COMPONENTS (IMPORTANT)
-----------------------------

1. PERFORMANCE ELEMENT (Agent Brain)
- Takes input: customer_id or query
- Calls tools:
    - fetch customer data
    - fetch transactions
    - run churn prediction
- Produces:
    - churn risk
    - insights
    - recommended actions

Use LangChain agent (latest syntax preferred: LCEL / Runnable / tools)

-----------------------------

2. LEARNING ELEMENT
- Improves system over time
- Updates based on:
    - customer feedback
    - success/failure of offers
- Can:
    - store interaction logs
    - update rules or weights
    - optionally retrain churn model (future scope)

-----------------------------

3. CRITIC
- Evaluates output quality
- Example:
    - Did customer accept offer?
    - Did churn reduce?
- Produces feedback signal:
    success / failure / score

-----------------------------

4. PROBLEM GENERATOR
- Suggests exploratory actions
- Example:
    - Try new type of offer
    - Try different retention strategy
- Encourages learning instead of static behavior

-----------------------------
